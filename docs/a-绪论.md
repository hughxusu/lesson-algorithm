# 绪论

尼克劳斯-沃思（Niklaus Wirth，瑞士计算机科学家，1984 年图灵奖得主），提出了计算机领域著名的公式
$$
\text{程序}=\text{数据结构}+\text{算法}
$$

* 数据结构：计算机存储、组织数据的方式。
* 算法：程序解决问题的步骤。

数据结构就像食材，算法就像菜谱，二者是共生关系，缺少任意一个都无法做出一道菜。

## 数据结构

数据结构（Data Structure）是计算机存储、组织数据的方式。
$$
\text{数据结构}=\text{存储}+\text{关系}
$$

* 存储：数据在内存中是怎么放的（挨着放还是随便找空位放）。
* 关系：数据和数据之间是什么关系（谁排在谁前面，谁是谁的上级）。

常见的几种数据结构：

1. 线性结构：一根绳上的蚂蚱。
2. 树形结构：开枝散叶。
3. 图形结构：错综复杂的关系网。

## 算法

算法是一系列定义清晰的指令，用于完成计算、处理数据或做出决策。算法的五个特性

* 有穷性（Finiteness）：算法不能无限循环，必须在执行有限个步骤后自动结束。
* 确定性（Definiteness）：每一步骤必须有明确的含义，不能含糊不清。
* 输入项（Input）： 有0个或多个输入。
* 输出项（Output）：至少有一个输出。
* 可行性（Effectiveness）：每一个步骤都必须是可以在现实中通过有限次运算实现的。

> [!note]
>
> 算法是独立存在的一种解决问题的方法和思想。

对于算法而言，实现的语言并不重要，重要的是思想。算法可以使用不同的编程语言实现，如：C实现、C++实现、Python实现等。

> [!tip]
>
> 如果$a+b+c=1000$，且$a^2+b^2=c^2$。$a,b,c$ 为自然数，求出所有$a,b,c$可能的组合?

对于解决问题没有头绪时，可以使用穷举法，来解决问题。

穷举法：将问题的所有可能的答案一一列举，然后根据条件判断此答案是否合适，合适就保留，不合适就丢弃。

```python
%%time
for a in range(1001):
    for b in range(1001):
        for c in range(1001):
            if a**2 + b**2 == c**2 and a + b + c == 1000:
                print(a, b, c)
```

### 人工智能算法

计算机数据结构与算法是通用的底层工具集，人工智能算法是基于这个工具集，针对特定智能任务设计的专用方法。

* 通用算法和数据结构是螺丝刀、锤子、扳手这些基础工具；
* 人工智能算法是用这些工具组装出来的机床或设备，专门用来完成特定的复杂任务。

|          | 计算机数据结构与算法                                         | 人工智能算法                                                 |
| -------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 目标     | 解决通用的、确定性的计算问题，追求高效性。                   | 解决非确定性的、智能的任务，如：识别、预测。追求任务准确率等。 |
| 处理对象 | 结构化的数据，如：数组、链表、树、图等，输入输出通常是明确的。 | 可以是结构化、半结构化、非结构化数据如：图像、文本、语音等，输入输出可能模糊、存在噪声。 |
| 核心思想 | 基于确定的逻辑和步骤，如：冒泡排序、二分查找等。             | 基于概率统计、迭代优化等，如：神经网络的反向传播、决策树的信息增益。 |
| 结果特性 | 结果是确定的、可重复的，相同输入必然得到相同输出             | 结果可能是概率性的、近似的，相同输入可能因初始化或噪声得到略有差异的输出 |
| 典型代表 | 贪心算法、动态规划、图算法等                                 | 神经网络、决策树、支持向量机（SVM）等                        |

人工智能算法依赖数据结构与算法作为基础

* 例如训练神经网络时，需要用**矩阵运算**（底层是数组的遍历与计算）；
* 构建决策树时，需要用递归算法划分特征，用树结构存储决策规则；
